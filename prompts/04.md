# Tarefa

Em @app\chat\page.tsx crie a interface da print anexada no prompt

**VOCÊ PRECISA** fazer a interface identica a print anexada **CUSTE O QUE CUSTAR**

# Requisitos técnicos

- Use a Vercel AI SDK e a rota "/api/chat" para construir o chat
- Apenas envie e receba as mensagens, não efetue nenhum tipo de formatação (vamos usar Streamdown depois)
- Use Context7 para buscar documentações da Versel AI SDK
- O botão de gravar áudio não deve fazer nada por enquanto
- deve ser possível enviar mensagem ao apertar enter
- adicione ao lado do botão de gravar áudio um botão para enviar a mensagem
